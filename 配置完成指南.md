1. **重启Cursor**（让配置生效）
2. 打开LeetCode插件
3. 选择一个题目
4. 点击 **"Code Now"**
5. 文件应该会自动创建在 `py/{题目编号}.{题目名称}/solution.py`

### 第二步：整理现有文件（可选）

如果你想整理现有的文件到新结构：

```bash
# 1. 先预览（不会实际移动文件）
python3 reorganize_structure.py --dry-run

# 2. 确认无误后执行（会自动备份）
python3 reorganize_structure.py --backup
```

这会：
- 将 `1.two-sum.py` → `py/1.two-sum/solution.py`
- 将 `1.two-sum.cpp` → `cpp/1.two-sum/solution.cpp`
- 自动创建README文件

## 🔧 配置说明

### 插件设置

已在 `.vscode/settings.json` 中配置：

```json
{
  "leetcode-problem-rating.defaultLanguage": "python3",
  "leetcode-problem-rating.filePath": {
    "python3": {
      "folder": "py/${id}.${kebab-case-name}",
      "filename": "solution.py"
    },
    "cpp": {
      "folder": "cpp/${id}.${kebab-case-name}",
      "filename": "solution.cpp"
    }
  }
}
```

### 切换默认语言

编辑 `.vscode/settings.json`：

```json
{
  "leetcode-problem-rating.defaultLanguage": "cpp"  // 改为cpp或其他语言
}
```

## 🆘 遇到问题？

### Q: 插件没有按配置路径创建文件？

A: 
1. 检查 `.vscode/settings.json` 配置是否正确
2. 重启Cursor
3. 检查插件版本是否支持该配置

### Q: 如何更改已创建题目的位置？

A: 
1. 手动移动文件到新位置
2. 或使用 `reorganize_structure.py` 脚本自动整理

### Q: 多解法文件如何管理？

A: 
- 推荐：每个解法一个文件
- 备选：在同一文件中用注释保留多个解法
