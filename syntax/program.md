# 语法学习规则文档

## 目录结构

```
syntax/
├── py/          # Python 语法笔记
├── cpp/         # C++ 语法笔记
├── java/        # Java 语法笔记
└── ...          # 其他语言
```

## 文件命名规则

1. **命名格式**：`题目编号.题目名称.ipynb`
2. **示例**：
   - `1.two-sum.ipynb`
   - `2.add-two-numbers.ipynb`
   - `9.palindrome-number.ipynb`

## 文件格式

- 使用 **Jupyter Notebook** (`.ipynb`) 格式记录 Python 语法
- 其他语言可以根据需要使用相应格式（如 Markdown）

## 内容要求

每个 notebook 应包含：
1. **题目关联**：记录与特定题目相关的语法知识点
2. **详细解释**：语法点的说明和原理
3. **代码示例**：可运行的示例代码
4. **实践应用**：在题目中的具体使用场景

## 维护规则

- 每遇到新的语法知识点，在对应的 notebook 中补充
- 保持命名与题目编号一致，方便查找
- 定期复习和整理语法笔记
